# Ziglana

A static website for the Ziglana community (Zig developers building AI and crypto on Solana).

## Technologies Used

*   HTML
*   CSS
*   JavaScript

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in your browser.

## Code Example

```zig
const std = @import("std");

pub fn main() !void {
    std.debug.print("Hello, Ziglana!\n", .{});
}
```
